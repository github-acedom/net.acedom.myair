Transform your home into an oasis of comfort with the myAir Control App, your companion for personalised air conditioning with Advantage Air myAir systems. This app gives Homey control over the main aircon unit and discovered zones, making it easy to manage comfort, airflow, and automations from one place.

Key features include zone-specific control, main unit on/off control, aircon mode and fan speed control, support for both temperature-controlled and vent-percentage zones, Homey Flow cards for automation, optional command throttling for controllers that do not like rapid consecutive commands, shared retry handling for polling and commands, and a communication-failure Flow trigger for detecting ongoing connection issues.
